# BongoCat

一个轻量的 Windows 桌面猫咪小工具 / 桌宠项目。  
当前仓库内容以**可直接运行的发布版文件**为主，包含主程序、卸载程序以及所需的模型资源文件，适合直接下载后体验。

---

## 项目简介

BongoCat 是一个面向 Windows 平台的小型桌面互动应用。  
从当前项目结构来看，程序集成了托盘图标资源与 Live2D 模型资源，可用于桌面陪伴、展示或轻量交互场景。

当前仓库适合：
- 直接下载运行
- 作为个人项目展示
- 后续继续补充源码、说明文档与截图

---

## 功能特点

- 支持 Windows 平台运行
- 提供可执行程序，开箱即用
- 包含托盘相关资源
- 集成模型资源文件
- 当前资源目录中包含 `standard` 与 `keyboard` 两套模型配置
- 适合作为桌宠/互动展示类小项目进行二次整理与展示

---

## 项目结构

```text
BongoCat/
├─ bongo-cat.exe          # 主程序
├─ uninstall.exe          # 卸载程序
└─ assets/
   ├─ tray.png            # 托盘图标资源
   └─ models/
      ├─ standard/        # 标准模型资源
      └─ keyboard/        # 键盘模式模型资源
```

## 运行方式

### 方式一：直接运行

1. 下载或克隆本仓库
2. 进入项目目录
3. 双击运行：

```bash
bongo-cat.exe
```

### 方式二：通过 Git 克隆

```bash
git clone https://github.com/Aprivity/BangoCat-.git
cd BangoCat-
```

然后直接运行 `bongo-cat.exe`。

---

## 使用说明

- 首次运行时，Windows 可能会弹出安全提示，这是未签名桌面程序较常见的情况。
- 如果程序启动后没有明显窗口，可以留意系统托盘区域是否已有图标。
- 如需移除程序，可使用目录中的 `uninstall.exe`。
